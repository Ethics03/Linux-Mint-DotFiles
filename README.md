# i3WM Dotfiles

Clone this repo with this command:

```bash
git clone https://github.com/Ethics03/Linut-Mint-Dotfiles.git
```
-> Make sure u configure your picom as i3wm dosent have
its own compositor

```
bspwm
nitrogen
polybar

kitty
fastfetch
rofi
sxhkd
synth-shell
picom
neofetch

```
